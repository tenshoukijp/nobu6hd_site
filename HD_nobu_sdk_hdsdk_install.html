  • HD.version.SDKとは？

    これは「天翔記HD版ソフトウェア開発キット」となります。

    プログラム経験が必要 プログラム用のSDKですので、ある程度はプログラムの心得があることが前提となります。
    プログラム未経験の方は対象とはしていません。

    主に出来ること

    • 戦略画面や戦争画面の様々な値の取得や書換

      武将・軍団・大名・家宝・官位・城・国・県・暦・戦略・戦争・
      音楽・演出・効果音・ダイアログ・環境設定

    • 読込ファイル名の変更

      ファイルを別にしておくことで、編集したファイルが更新当の影響を受けるのを回避出来る

    • メッセージの変更

      元のメッセージを状況に応じて新メッセージへと振り分け
      人物関係に応じた自動語句変化用API付

    • 顔グラ変更

      取込画像への変更含む

    • 音楽変更、新たなタイミングで音楽や効果音を再生

      状況に応じて13曲ではなく150曲に振り分けるなど自由

    • 簡易なイベントの作成

      ダイアログ・フェイドアウト・フェイドイン・カメラ等

    これらを元のデータファイルを一切変更することなく行うことが出来ます。

  • 動作環境

    %(environment_os)s %(environment_dnet)s
    開発環境 Visual Studio 2015 Update3 以上 より新しい開発環境でも問題は無いと思われます。
    使用言語 ユーザー編集対象部分は、C++/CLIのCLI寄りとなります。

  • 使い方

    • ファイルの置き場所と実行方法

      version.dllとHD.version.dllを天翔記ディレクトリへと配置。
      天翔記HD版を実行。

    • HD.version.dll

      ご自身でVisual Studio 2015を使い、
      「天翔記クラス.cpp」などのソースへ「自分で行いたいことを」を記述。
      コンパイルしてdllを作成することとなります。

    注意点

      コンパイル済みHD.version.dllのバイナリは決して配布しないで下さい!!
      バイナリとSDKが競合してどちらか一方が使えなくなり、混乱の元となるためです。
      配布は必ず「ソースで」行ってください。

    関連ツール

    • %(ts95)s用のScenarioMod

      %(ts95)s用として、ScenarioModという本SDKに類似したものを提供しています。

      「%(ts95)s用のScenarioMod」と「%(tshd)s用のHD.version.SDK」では、
      ScenarioModの方が約２倍ほど出来ることが多くなっています。
      特に「ユーザー独自に新たなイベントを作成する」という部分ではAPI提供数に大きな開きがあります。

      しかし、HD.version.SDKは、現代的な.NET Framework の考え方に沿っており、
      ScnearioModの反省点も踏まえ、構成全体がよく整理されています。

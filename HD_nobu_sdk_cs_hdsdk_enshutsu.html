User.version.SDK ～演出～

演出の概念

フェイドインやフェイドアウト、画面カメラの移動など、
画面演出系のAPI提供用オブジェクトとなります。

使い処

フェイドアウトやフェイドイン、画面中央に映す城を変更する際に利用します。

使用例

フェイドアウト・カメラ移動・フェイドイン 
    public static void My_戦略_軍団ターン変更時(int 軍団番号)
    {
        演出.Do_フェイドアウト();
        if (軍団.配列.Exists(軍団番号))
        {
            int iCastleID = 軍団.配列[軍団番号].所属城番号;

            // カメラ移動とは、その城を出来るだけ画面の真ん中に映すカメラへと移動する。
            演出.Do_カメラ移動(iCastleID);
        }
        演出.Do_フェイドイン();
    }

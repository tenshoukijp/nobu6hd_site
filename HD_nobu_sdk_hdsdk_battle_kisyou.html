  • 戦争->気象の概念

    戦争->気象とは、戦争画面の天気を扱うオブジェクトのことです。

  • 使用例①

    現在の所、戦争->気象オブジェクトが提供するものは、
    戦争画面の「天気」情報のみの提供となります。

    天気の情報を得る 
    String^ 天翔記クラス::On_噴出メッセージ直前(String^ 元メッセージ, 噴出メッセージパラメタ型^ パラメタ) {

  if (戦争->Is_野戦中 || 戦争->Is_籠城戦中) {
    if (戦争->気象->天候 == 列挙::戦争::気象::天候::晴れ) {
      return "晴れです。";
    }
    else if (戦争->気象->天候 == 列挙::戦争::気象::天候::曇り) {
      return "曇りです。";
    }
    if (戦争->気象->天候 == 列挙::戦争::気象::天候::雨) {
      return "雨です。";
    }
    if (戦争->気象->天候 == 列挙::戦争::気象::天候::雪) {
      return "雪です。";
    }
  }
  return "";
}

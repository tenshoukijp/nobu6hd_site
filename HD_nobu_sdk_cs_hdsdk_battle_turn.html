User.version.SDK ～戦争.ターン～

  • 戦争.ターンの概念

    戦争.ターンとは、戦争画面の様々ターンを扱うオブジェクトのことです。

  • 使用例①

    現在の所、戦争.ターンオブジェクトが提供するものは、
    戦争画面の「残りターン数」情報のみの提供となります。
    「My_戦争_残りターン変更時」のメソッドの「残りターン数」の情報を、
    どのイベントハンドラでも利用可能にしていると考えれば良いでしょう。

    戦争時の残りターン数を得る 
        public static string My_噴出メッセージ直前(string 元メッセージ, 噴出メッセージパラメタ型 パラメタ)
    {
        if (戦争.Is_野戦中 || 戦争.Is_籠城戦中)
        {
            デバッグ出力("残りターン:" + 戦争.ターン.残りターン);
        }

        return "";
    }

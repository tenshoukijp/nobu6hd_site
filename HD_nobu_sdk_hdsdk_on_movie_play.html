On_ＭＯＶＩＥ再生直前

Movieが再生される直前に、
呼びだされるイベントハンドラとなります。

注意点

用意する新しいムービーは、オリジナルのムービーと再生時間の尺が同じか、
もしくは短い必要があります。
長い場合は、再生途中で再生が打ち切られることがあります。

使い処

元のオリジナルのファイル内容を差し替えるのではなく、
ムービーを新しいファイル名で用意して、そちらを再生したいようにする場合に利用します。

使用例①

このイベントハンドラでは、返り値を文字列で返すと、
元のファイルの替わりに、指定のファイルを再生します。

もしも、該当のファイルが存在しない場合、
Movieは再生されません。

「ＭＯＶＩＥ番号」はあまり意味を成す番号ではありませんので、
通常は、「元ファイル名」によって判定することになるかと思います。

オープニングムービーの替わりに、自分で用意したムービーを再生する
String^ 天翔記クラス::On_ＭＯＶＩＥ再生直前(int ＭＯＶＩＥ番号, String^ 元ファイル名) {
  if (元ファイル名->ToLower() == "op.n6") {
    return "my_OP.N6";
  }
  else if (元ファイル名->ToLower() == "op_l.n6") {
    return "my_OP_L.N6";
  }
  return "";
}

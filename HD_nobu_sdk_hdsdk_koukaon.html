HD.version.SDK ～効果音～

  • 効果音の概念

    各種、天翔記内で使用されている効果音を、
    好きなタイミングで再生するためのAPI用オブジェクトとなります。

  • 使い処

    独自に何らかの効果音を再生したい場合に利用します。

  • 使用例

    メソッド 
    void 天翔記クラス::On_戦略_軍団ターン変更時(int 軍団番号) {
    // 夏になった際の効果音を再生する。
    効果音->Do_効果音再生(列挙::効果音::効果音音源::夏);
}

  • その他

    効果音に関する主な所は以上となります。
    ゲーム内には約40種類の効果音が用意されていますので、
    「効果音.h」や「効果音列挙.h」などを参照してください。
